# homebrew-shikomi

Homebrew tap for [shikomi](https://github.com/shikomi-dev/shikomi) — global hotkey clipboard manager.

## Install

```bash
brew tap shikomi-dev/homebrew-shikomi
brew install --cask shikomi
```

## Supported platforms

- macOS 12 Monterey or later (Apple Silicon / arm64)

> **Note**: v0.1.0 is an unsigned build (MVP phase). macOS Gatekeeper will show a warning on first launch.
> Right-click the app → Open to bypass. Code signing is tracked in [Issue #130](https://github.com/shikomi-dev/shikomi/issues/130).
